# 🚀 Day 01 – My DevOps Learning Plan

## My Understanding of DevOps

DevOps is more than just tools—it's a way of working that brings development and operations together. It focuses on collaboration, automation, continuous integration, continuous delivery (CI/CD), and monitoring to build and deploy software faster and more reliably.

## Why I'm Learning DevOps & Cloud

I'm currently pursuing my MCA and want to build practical skills that companies actually look for. DevOps interests me because it combines Linux, Cloud, Networking, Automation, Containers, and Infrastructure into one career path.

My goal isn't just to watch tutorials. I want to learn by doing, build real projects, and document my progress throughout this 90-day journey.

## My End Goal

By the end of these 90 days, I want to have strong practical knowledge, a solid GitHub portfolio, and the confidence to apply for DevOps Engineer roles.

## Current Level

* Basic knowledge of Linux, Networking, Git, Shell Scripting, and AWS
* Starting my structured DevOps journey

## Goals for the Next 90 Days

## Core DevOps Skills to Build
